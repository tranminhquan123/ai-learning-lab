# AI Learning Lab

Repo này dùng để quản lý hành trình học AI Engineer / AI Application Developer, Claude AI workflow, tài chính - đầu tư, tiếng Trung và DaVinci Resolve.

Roadmap Python cơ bản được quản lý riêng, repo này chỉ tập trung vào các phần sau Python nền tảng và các kỹ năng bổ trợ.

## Mục tiêu chính

- Xây dựng nền tảng AI/ML/LLM.
- Học cách build AI application bằng Python.
- Thiết lập Claude Code, MCP, memory và workflow học tập cá nhân.
- Học tài chính - đầu tư từ nguồn đáng tin cậy.
- Học tiếng Trung phục vụ công việc.
- Học DaVinci Resolve để edit video.

## Cấu trúc repo

- `roadmap/`: lộ trình học từng mảng.
- `notes/`: ghi chú kiến thức.
- `projects/`: project thực hành.
- `prompts/`: prompt dùng lại với Claude.
- `skills/`: Claude skills cá nhân.
- `memory/`: tiến độ học, lỗi hay gặp, phần đã nắm.
- `resources/`: tài liệu, sách, repo, paper, nguồn học.
- `configs/`: cấu hình mẫu cho Claude Code, MCP, môi trường làm việc.
