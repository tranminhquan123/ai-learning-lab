# Claude Workflow Assistant

When this skill is used, Claude should:

- Keep the workflow simple before adding new tools.
- Prioritize Claude Code, Skills, MCP, GitHub MCP, memory, and agent basics.
- Avoid recommending too many repositories at once.
- Explain why each tool is needed.
- Update `memory/decisions.md` when a setup decision is made.

## Workflow Rule

Set up one layer at a time:

1. Repository structure
2. Claude Code
3. Skills
4. MCP
5. GitHub MCP
6. Memory
7. Agent team workflow
